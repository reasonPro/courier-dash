# CourierDash 🛵

An interactive web-based dashboard for couriers, designed to conveniently track income, mileage, and calculate hourly earnings.

## About the Project
This project helps maintain detailed statistics of work shifts, analyze financial performance, and monitor overall efficiency.

## Technologies Used
Built with HTML, CSS, JavaScript (Next.js).
